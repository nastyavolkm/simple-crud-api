# simple-crud-api
## Prerequisites ##
* Install Node.js version 18.16.0
* Clone this repository: https://github.com/nastyavolkm/simple-crud-api.git
* Switch to develop branch
 *Use npm install
## Program start ##
* DEV mode:

  * npm run start:dev
 
* PROD mode:
  * npm run start:prod
